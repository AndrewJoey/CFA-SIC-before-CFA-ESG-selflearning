## 什么是 CFA-SIC？

**CFA Institute 可持续投资与气候证书（CFA-SIC）** 是由全球顶级金融认证机构 CFA Institute 颁发的专业证书。它由原来的 **CFA ESG 投资证书** 更名而来。

### 为什么要考这个证书？

| 优势 | 说明 |
|---|---|
| **行业认可度高** | 由 CFA Institute 颁发，在全球金融行业具有极高权威性 |
| **职业发展需求** | 资产管理、银行、企业融资等领域对 ESG 能力的需求持续增长 |
| **零门槛入门** | 无任何报考前提条件，金融、法律、咨询或非金融背景人士均可报考 |
| **实用性强** | 涵盖真实场景下的 ESG 分析、投资组合整合与股东尽责管理 |

### 适合哪些人？

- 希望补充 ESG 专业技能的金融从业者
- 准备进入投资行业的在校学生
- 从事企业可持续发展工作、希望向金融领域转型的专业人士
- 对"环境、社会与治理因素如何影响投资决策"感兴趣的任何人

## 考试概览

| 项目 | 说明 |
|---|---|
| **题型** | 100 道单选题 |
| **考试时长** | 约 2 小时 20 分钟 |
| **考试方式** | 在线监考（可在家或在考试中心参加） |
| **及格分数** | 官方未公布，候选人社区普遍引用约 60% |
| **补考政策** | 允许多次报考，具体规则请以 CFA Institute 官网为准 |
| **考试语言** | 英语 |
| **报名方式** | 通过 CFA Institute 官方网站报名 |

> **零基础提示：** 考试侧重理解与应用，而非死记硬背。重点理解 *为什么* ESG 因素对投资重要，以及 *如何* 在分析中应用它们。

## 考试大纲与各模块

课程共分为 **7 个模块**，以下为各模块的大致考察权重：

| 序号 | 模块名称 | 大致权重 |
|---|---|---|
| 1 | ESG 投资简介 | ~10% |
| 2 | 环境因素 | ~15–20% |
| 3 | 社会因素 | ~15–20% |
| 4 | 治理因素 | ~15–20% |
| 5 | 股东参与与尽责管理 | ~10% |
| 6 | ESG 分析、估值与整合 | ~15% |
| 7 | ESG 整合型投资组合构建与管理 | ~10–15% |

### 各模块内容简介

**模块 1 — ESG 投资简介**
定义 ESG 概念，解释其对投资者的意义，介绍主要参与方（资产所有者、资产管理人、发行人、监管机构）。梳理从社会责任投资（SRI）到现代 ESG 整合的发展历程。

**模块 2 — 环境因素**
涵盖气候变化、碳排放、能源转型、水资源短缺、生物多样性，以及企业面临的物理风险与转型风险。介绍气候相关财务信息披露工作组（TCFD）等重要框架。

**模块 3 — 社会因素**
研究劳工实践、供应链标准、人权、产品安全、社区关系与数据隐私。阐述社会风险如何影响企业声誉、法律责任和长期业绩。

**模块 4 — 治理因素**
涵盖董事会结构与独立性、高管薪酬、股东权利、审计质量、反腐败政策与企业文化。治理是 ESG 三大支柱中最可量化的，也是考试考点最集中的模块之一。

**模块 5 — 股东参与与尽责管理**
讲解投资者如何通过积极所有权影响企业行为：包括对股东提案投票、与管理层开展直接沟通以及参与协作倡议等。

**模块 6 — ESG 分析、估值与整合**
教授如何将 ESG 数据纳入传统财务分析和估值模型。涵盖 ESG 评分方法论、评级机构及其局限性。

**模块 7 — ESG 整合型投资组合构建与管理**
讲解 ESG 在不同资产类别（股票、固定收益、实物资产、另类资产）中的应用，以及如何制定体现 ESG 目标的投资授权。

## 学习计划

本计划专为**零基础学员**设计，无需金融或 ESG 基础知识，可根据个人时间安排灵活调整进度。

> **预计总学习时长：** 8–12 周，共 80–120 小时

### 8 周学习路线图

```
第 1-2 周  │  基础入门
           │  ├─ 模块 1：ESG 投资简介
           │  ├─ 建立 ESG 基础词汇体系
           │  └─ 阅读入门级 ESG 科普文章

第 3-4 周  │  环境与社会因素
           │  ├─ 模块 2：环境因素（TCFD、气候风险）
           │  └─ 模块 3：社会因素（劳工、供应链、隐私）

第 5-6 周  │  治理与尽责管理
           │  ├─ 模块 4：治理因素
           │  └─ 模块 5：股东参与与尽责管理

第 7 周    │  分析与投资组合整合
           │  ├─ 模块 6：ESG 分析与估值
           │  └─ 模块 7：ESG 整合型投资组合构建

第 8 周    │  复习与冲刺
           │  ├─ 全面回顾薄弱环节
           │  ├─ 刷题（建议完成 200+ 道练习题）
           │  └─ 模拟限时考试，熟悉考场节奏
```

### 每周学习建议

| 周次 | 学习重点 | 建议学时 |
|---|---|---|
| 第 1 周 | 阅读模块 1，建立 ESG 词汇体系 | 8–10 小时 |
| 第 2 周 | 完成模块 1 练习题，回顾术语 | 6–8 小时 |
| 第 3 周 | 模块 2（环境因素） | 10–12 小时 |
| 第 4 周 | 模块 3（社会因素） | 10–12 小时 |
| 第 5 周 | 模块 4（治理因素） | 10–12 小时 |
| 第 6 周 | 模块 5（尽责管理） | 8–10 小时 |
| 第 7 周 | 模块 6–7（分析与投资组合） | 12–15 小时 |
| 第 8 周 | 全面复习 + 模拟考试 | 10–12 小时 |

### 零基础学员备考建议

- **先理解"为什么"：** 在深入细节之前，先弄清楚 ESG 为何在金融行业变得如此重要。有了宏观视角，后续内容会更容易理解和记忆。
- **重视治理模块：** 模块 4 通常是考点最密集的部分，建议投入更多时间。
- **结合真实案例学习：** 学到某个概念（如碳转型风险）时，不妨搜索一则相关的企业新闻。将抽象概念与现实事件对应，记忆效果事半功倍。
- **多做题，少重读：** 每个模块通读一遍后，建议切换到刷题模式，而不是反复阅读同一段文字。
