# 模块 3：社会因素 | Module 3: Social Factors

> **考试权重：~15–20%** | 社会因素是 ESG 三个支柱中最"与人相关"的维度。
>
> 核心关注企业与员工、供应链、社区、客户之间的关系管理。

## 1. 社会因素的范畴

### 1.1 S 维度涵盖什么？

**社会因素**关注企业与"人"的关系——包括内部的人和外部的人：

| 利益相关者群体 | 关注议题 |
|---|---|
| **员工** | 劳动条件、薪酬公平、健康与安全、DEI、培训与发展 |
| **供应链工人** | 强迫劳动、童工、工资拖欠、工作环境安全 |
| **客户/消费者** | 产品安全、数据隐私、公平营销、可及性 |
| **社区** | 社区关系、原住民权利、当地就业、环境影响 |
| **社会/公众** | 人权、税收公平、政治游说、社会贡献 |

### 1.2 利益相关者 vs 股东（Stakeholder vs Shareholder）

| 维度 | 股东优先模型 | 利益相关者模型 |
|---|---|---|
| 核心关注 | 最大化股东价值 | 平衡所有利益相关者的利益 |
| 决策依据 | 财务回报 | 财务 + 社会 + 环境综合考量 |
| 风险视角 | 财务风险 | 包含社会声誉风险、经营许可风险 |

> **💡 ESG 投资天然倾向于利益相关者模型**，因为社会因素的外部性最终会转化为财务风险。

## 2. 劳工标准与人权 ⭐

### 2.1 国际劳工组织（ILO）核心公约

ILO 于 1998 年通过《工作中的基本原则和权利宣言》，确立四大基本原则：

| 原则 | 说明 |
|---|---|
| **结社自由与集体谈判权** | 工人有权组建工会和进行集体谈判 |
| **消除强迫劳动** | 禁止所有形式的强迫或强制劳动 |
| **有效废除童工** | 禁止使用童工，设定最低工作年龄 |
| **消除就业歧视** | 禁止基于种族、性别、宗教等的就业歧视 |

> **⚠️ 考试提示：** ILO 四大原则 = 结社自由、无强迫劳动、无童工、无歧视。这四个是必记的。

### 2.2 现代奴役（Modern Slavery）

**定义：** 现代奴役涵盖强迫劳动、债役、人口贩运、强迫婚姻等。供应链是现代奴役风险的**重点领域**。

| 风险指标（Red Flags） | 说明 |
|---|---|
| 招聘费过高 | 工人被收取高额中介费，被迫以工作偿还债务 |
| 扣留护照/身份证件 | 雇主控制工人的旅行自由 |
| 低于最低工资 | 工资水平违法且不足以维生 |
| 过度加班 | 强制超时工作（尤其在电子、服装行业） |
| 不安全工作条件 | 缺乏基本安全保障（如建筑、采矿行业） |
| 限制人身自由 | 工人不能自由离开工作场所或宿舍 |

**立法动态：**
- 英国《现代奴役法案》（2015）：要求企业发布年度现代奴役声明
- 澳大利亚《现代奴役法案》（2018）：类似要求
- 欧盟《企业可持续发展尽职调查指令》（CSDDD）：强制企业进行人权和环境的供应链尽职调查

### 2.3 联合国商业与人权指导原则（UNGPs）

三大支柱框架：
1. **国家有义务保护人权**（State duty to protect）
2. **企业有责任尊重人权**（Corporate responsibility to respect）
3. **受害者应获得有效救济**（Access to remedy）

## 3. 供应链管理

### 3.1 为什么供应链是社会因素的核心？

大多数社会风险（尤其是劳工和人权风险）发生在**一级供应商之外**的更上游供应链中。企业面临的供应链挑战：

| 挑战 | 说明 |
|---|---|
| **供应链透明度不足** | 多级供应商结构复杂，难以追溯到原材料来源 |
| **审计局限性** | 定期审计可能无法发现系统性问题，供应商可能"应付审计" |
| **采购压力传导** | 品牌方不断压价和缩短交期 → 供应商压低工资、强制加班 |
| **冲突矿产** | 刚果等地矿产（锡、钽、钨、金）开采与武装冲突关联 |

### 3.2 供应链尽职调查步骤

```
第1步：识别 → 绘制供应链地图，定位高风险区域/环节
第2步：评估 → 对识别出的风险进行严重性和可能性评估
第3步：应对 → 采取纠正措施（培训供应商、修改合同条款、终止合作）
第4步：追踪 → 持续监测改善效果
第5步：沟通 → 公开报告尽职调查过程和结果
第6步：救济 → 为受害者提供补救渠道
```

> **⚠️ OECD 跨国企业准则**和**联合国工商企业与人权指导原则**是供应链尽职调查的两个基础性国际框架。

## 4. 人力资本管理

### 4.1 DEI（多元化、公平与包容）

| 维度 | 定义 | 衡量指标 |
|---|---|---|
| **多元化**（Diversity） | 员工在性别、种族、年龄、背景等方面的多样性 | 性别比例、少数族裔占比、年龄分布 |
| **公平**（Equity） | 确保所有人获得公平的机会和待遇 | 薪酬差距、晋升率、入职门槛 |
| **包容**（Inclusion） | 所有员工都感到被尊重和能参与的文化氛围 | 员工满意度调查、离职率 |

**投资者关注的 DEI 议题：**
- 董事会性别多样性（很多法域有最低比例要求，如欧盟 40%）
- 性别薪酬差距（Gender Pay Gap）报告
- 种族/族裔薪酬公平

### 4.2 员工健康与安全

| 指标 | 说明 |
|---|---|
| **工伤率**（LTIFR — Lost Time Injury Frequency Rate） | 每百万工时中导致缺勤的工伤次数 |
| **死亡事故**（Fatalities） | 工作中死亡人数 |
| **职业病** | 长期暴露导致的健康损害 |

### 4.3 人力资本开发

- 培训与发展投入（培训时数/人、培训支出占薪酬比例）
- 员工敬业度调查得分
- 内部晋升率 vs 外部招聘率
- 员工流失率（Turnover Rate）

## 5. 产品责任与消费者保护

### 5.1 产品安全

| 行业 | 典型社会风险 |
|---|---|
| **制药** | 药品副作用、临床试验透明度、定价公平性 |
| **食品饮料** | 食品安全、标签准确性、营养成分 |
| **汽车** | 车辆安全、排放数据真实（大众柴油门事件） |
| **消费品** | 产品缺陷召回、化学成分安全 |
| **金融** | 不当销售（mis-selling）、掠夺性贷款、收费透明度 |

### 5.2 数据隐私与网络安全

在数字化时代，数据隐私已成为核心社会议题：

| 关键法规 | 地区 |
|---|---|
| **GDPR**（通用数据保护条例） | 欧盟 |
| **CCPA**（加州消费者隐私法） | 美国加州 |
| **个人信息保护法**（PIPL） | 中国 |

**投资视角：** 数据泄露不仅带来监管罚款，更导致客户流失和声誉损害。

## 6. 社区关系与经营许可

### 6.1 经营的社会许可（Social License to Operate）⭐

> **定义：** 企业运营获得当地社区和社会的**持续认可**。它不是政府颁发的正式许可证，而是一种"非正式的社会契约"。

| 有社会许可 | 失去社会许可 |
|---|---|
| 社区支持、员工自豪、监管宽松 | 抗议、诉讼、政府吊销营业执照 |

**典型案例：**
- 矿业项目因未征得原住民同意被叫停
- 化工厂污染引发周边居民抗议和环境诉讼

### 6.2 利益相关者参与（Stakeholder Engagement）

有效的社区关系管理需要**持续、真诚的利益相关者沟通**：
- **识别**：绘制所有受影响的利益相关者群体
- **沟通**：定期对话，不只"告知"更要"倾听"
- **回应**：根据反馈调整运营做法
- **报告**：公开披露社区参与的结果

### 6.3 原住民权利

联合国《原住民权利宣言》（UNDRIP）确立**自由、事先和知情同意（FPIC）** 原则：
- 在原住民土地上开发项目前，必须获得其自愿同意
- 矿业、能源、林业是最常涉及 FPIC 的行业

## 7. 社会因素与财务影响的关联

### 7.1 社会风险如何转化为财务风险？

| 社会风险事件 | 财务影响 |
|---|---|
| 供应链童工丑闻暴露 | 消费者抵制 → 营收下降；监管罚款 |
| 数据泄露 | 罚款（GDPR 可达全球营收 4%）+ 客户流失 |
| 重大工伤事故 | 停产整顿 + 赔偿 + 保险费上浮 |
| 歧视诉讼 | 法律费用 + 赔偿金 + 雇主品牌受损 |
| 社区抗议 | 项目延期/取消 + 投资损失 |

### 7.2 行业社会风险矩阵

| 行业 | 关键社会风险 |
|---|---|
| **纺织/服装** | 供应链劳工、工资拖欠、建筑安全 |
| **电子/科技** | 冲突矿产、加班文化、数据隐私 |
| **食品饮料** | 食品安全、营养健康、供应链人权 |
| **制药** | 药品定价、临床试验伦理、可及性 |
| **金融** | 消费者保护、普惠金融、不当销售 |
| **采矿/能源** | 社区关系、原住民权利、工人安全 |
