# 模块 4：治理因素 | Module 4: Governance Factors

> **考试权重：~15–20%** | 治理是 ESG 三大支柱中**考点最密集**的模块。
>
> 作为最"可量化"的 ESG 维度，治理因素在投资分析中有悠久历史。

完成本模块后，你应该能够：

6. 比较不同公司所有权结构对治理的影响

## 1. 什么是公司治理？

### 1.1 定义

> **公司治理（Corporate Governance）** 是规范公司运作、决策、监督和问责的制度框架，旨在平衡管理层、董事会、股东和其他利益相关者的利益。

### 1.2 治理为什么是 ESG 的"基石"？

```
          ┌──────────────┐
          │  治理 (G)     │ ← 制度基础：没有良好治理，E 和 S 无从谈起
          ├──────────────┤
          │  环境 (E)  社会 (S)  │ ← 在良好治理基础上实现
          └──────────────┘
```

**代理问题（Agency Problem）：** 治理的核心关切。当管理层（代理人）的利益与股东（委托人）的利益不一致时产生的问题。

- 股东追求长期价值最大化
- 管理层可能追求短期业绩（奖金驱动）、扩大帝国（并购）、规避风险（安逸）

> **💡 治理的设计目的就是缩小代理问题带来的利益偏差。**

## 2. 董事会结构与独立性 ⭐⭐⭐

### 2.1 单层 vs 双层董事会

| 模式 | 结构 | 常见地区 |
|---|---|---|
| **单层制**（One-tier / Unitary Board） | 执行董事 + 非执行董事在同一董事会中 | 美国、英国 |
| **双层制**（Two-tier Board） | 管理委员会 + 独立监事会 | 德国、荷兰、中国 |

### 2.2 董事类型

| 类型 | 英文 | 定义 |
|---|---|---|
| **执行董事** | Executive Director | 在公司担任管理职务（CEO、CFO 等）的董事会成员 |
| **非执行董事** | Non-Executive Director (NED) | 不在公司担任管理职务的董事会成员 |
| **独立非执行董事** | Independent Non-Executive Director (INED) | 与公司无任何实质利益关系（财务、家族、商业往来）的非执行董事 |

### 2.3 董事会独立性（必考核心！）⭐

**为什么独立性重要？**
- 独立董事能**客观监督**管理层，减少代理问题
- 在薪酬制定、审计、关联交易等敏感议题上，独立董事的参与是保障

**独立性判断标准：**
| 可能损害独立性的因素 | 说明 |
|---|---|
| 曾任公司或关联公司雇员 | 通常要求离职满一定年限（如 3-5 年） |
| 与公司有重大商业关系 | 如身为主要供应商的高管 |
| 亲属关系 | 与公司管理层或大股东有近亲关系 |
| 长期任职 | 任职时间过长可能导致"被同化"（通常 >9-12年有争议） |
| 交叉董事 | A 公司董事在 B 公司任职，B 公司董事在 A 公司任职 |

> **⚠️ 考试提示：** 题目常让判断一个董事是否是"独立"的。关键是看是否有**实质利益关系**。

### 2.4 董事会委员会

| 委员会 | 主要职责 | 独立性要求 |
|---|---|---|
| **审计委员会**（Audit Committee） | 监督财务报告和内部控制，与外部审计师对接 | 全部为独立董事（最常见的要求） |
| **薪酬委员会**（Remuneration Committee） | 制定高管薪酬方案 | 全部或多数为独立董事 |
| **提名委员会**（Nomination Committee） | 推荐董事会候选人，规划继任 | 多数为独立董事 |
| **风险委员会**（Risk Committee） | 监督风险管理框架（尤其金融业） | 独立董事参与 |

### 2.5 董事会多元化

| 维度 | 常见投资者关注点 |
|---|---|
| **性别多元化** | 女性董事比例（欧盟要求至少 40%） |
| **技能多元化** | 是否有财务、行业、ESG、数字化等关键技能的均衡配置 |
| **任期/年龄多元化** | 避免"全员老龄化"或"集体到期" |
| **种族/族裔多元化** | 美国纳斯达克和 SEC 有相关披露要求 |

## 3. 高管薪酬 ⭐⭐

### 3.1 薪酬结构的 ESG 考量

| 薪酬组成部分 | ESG 相关治理问题 |
|---|---|
| **基本工资**（Base Salary） | 是否合理，与行业对比 |
| **年度奖金**（Annual Bonus） | 是否仅绑定短期财务指标 |
| **长期激励**（LTIP — Long-Term Incentive Plan） | 是否包含 ESG 指标，业绩期是否足够长 |
| **股权激励** | 是否导致过度关注股价而忽视长期 ESG 风险 |

### 3.2 薪酬与 ESG 挂钩 ⭐

将 ESG 指标纳入高管薪酬是近年治理趋势：

| 做法 | 说明 | 争议 |
|---|---|---|
| 5-20% 奖金与 ESG 指标挂钩 | 最常见做法 | 指标选择可能过于简单（"设定了减排目标"就达标，而非实际减排） |
| 长期激励中加入 ESG 条件 | 3-5 年 ESG 目标 | 目标设定困难，初期缺乏标准化 |
| 碳减排目标绑定薪酬 | 石油天然气公司渐多采用 | "真实性"存疑 — 是否真能约束高管行为 |

### 3.3 薪酬话语权（Say-on-Pay）

> **Say-on-Pay** 是股东对高管薪酬方案进行**投票表决**的权利。

| 类型 | 说明 |
|---|---|
| **有约束力投票**（Binding Vote） | 提案被否决后薪酬方案必须修改 |
| **咨询性投票**（Advisory Vote） | 提案被否决后董事会可以不修改（但面临声誉压力） |
| **事后投票**（Ex-post Vote） | 对上一年薪酬执行情况进行表决 |

### 3.4 薪酬比（Pay Ratio）

CEO 薪酬与员工薪酬中位数的比值，是衡量薪酬公平性的指标。美国 SEC 自 2018 年起要求上市公司披露此指标。

## 4. 股东权利 ⭐⭐

### 4.1 一股一票原则

| 投票权安排 | 说明 | 治理关切 |
|---|---|---|
| **一股一票**（One Share, One Vote） | 每股有一票表决权 | 最佳实践 |
| **多重投票权**（Dual-Class Shares） | A 类股（多票）vs B 类股（少票），常见于科技公司 | 创始人控制过强，外部股东权利被稀释 |
| **投票权上限**（Voting Caps） | 限制任何股东的投票比例上限 | 保护中小股东但可能保护低效管理层 |

### 4.2 股东提案权

股东可在年度股东大会上提交提案，主题包括：
- 气候变化相关披露与策略
- 董事会多元化
- 政治献金透明度
- 高管薪酬

### 4.3 关联交易（Related Party Transactions）

与股东、董事或管理层有关联的第三方进行的交易。须：
- 独立董事审查批准
- 充分披露
- 公平定价

## 5. 审计质量与财务透明度

### 5.1 审计的核心治理要素

| 要素 | 最佳实践 |
|---|---|
| **审计师独立性** | 审计与非审计服务分离；定期轮换审计合伙人/事务所 |
| **审计委员会** | 全部由独立董事组成；至少一名财务专家 |
| **内部审计** | 直接向审计委员会汇报，不能仅向 CFO 汇报 |
| **审计意见** | 理想为无保留意见（Unqualified Opinion） |

### 5.2 财务报告透明度

**警示信号（Red Flags）：**
- 频繁更换审计师
- 持续经营的重大不确定性（going concern）
- 频繁发布财务重述（restatement）
- 关联交易缺乏披露
- 表外实体、复杂的公司结构
- 异常激进的会计政策

## 6. 反腐败与商业道德

### 6.1 贿赂与腐败

| 关键法规 | 管辖范围 |
|---|---|
| **美国《反海外腐败法》**（FCPA） | 禁止美国公司和个人贿赂外国官员 |
| **英国《反贿赂法》**（UK Bribery Act） | 全球最严格的反腐败法之一，涵盖商业贿赂 |
| **OECD 反贿赂公约** | 44 个签署国共同承诺打击跨国贿赂 |

### 6.2 反腐败最佳实践

- 制定并实施**反腐败政策**
- 建立**举报人保护机制**（Whistleblower Protection）
- 对高风险市场的第三方中介进行**尽职调查**
- 定期进行反腐败**培训**
- 公开报告违反行为准则的案例和处理结果

### 6.3 政治献金透明度

投资者越来越要求企业披露：
- 直接和间接（通过行业协会）的政治献金
- 游说支出
- 与行业贸易协会在气候政策上的立场是否一致

## 7. 所有权结构与控制

### 7.1 股权结构类型

| 结构 | 特征 | 治理优势 | 治理风险 |
|---|---|---|---|
| **分散型**（Dispersed） | 大量中小股东，无控股股东 | 股东民主 | 管理层控制（经理人专政）、搭便车 |
| **集中型**（Concentrated） | 一个或几个大股东控股 | 大股东有动力监督 | 大股东与小股东利益冲突、隧道挖掘 |
| **国有**（State-Owned） | 政府为控股股东 | — | 政策目标干扰商业决策、透明度低 |
| **家族控制**（Family-Controlled） | 家族控股 | 长期导向 | 接班人风险、裙带关系 |

## 8. 公司治理守则

### 8.1 各国治理守则概览

| 国家/地区 | 治理守则 | 特征 |
|---|---|---|
| **英国** | UK Corporate Governance Code | "遵守或解释"（Comply or Explain）模式的开创者 |
| **美国** | 无统一守则，SEC 规则+交易所规则 | 法律强制性更强 |
| **OECD** | G20/OECD 公司治理原则 | 国际基准，被广泛引用 |

### 8.2 "遵守或解释"原则（Comply or Explain）

> 公司**原则上应遵守**治理守则的每条规定，但如果不遵守，需公开**解释原因**。这种柔性方式给予企业灵活性，但要求透明度。

### 📝 关键定义速查

| 术语 | 一句话定义 |
|---|---|
| 代理问题 | 管理层与股东利益不一致的情况 |
| 独立董事 | 与公司无实质利益关系的非执行董事 |
| Say-on-Pay | 股东对高管薪酬方案的投票权 |
| 一股一票 | 每股拥有平等的表决权 |
| 隧道挖掘 | 控股股东将公司利益转移给自己（Tunneling） |
| 关联交易 | 与关联方进行的可能影响公司利益公平性的交易 |
| Comply or Explain | 遵守治理守则，否则公开解释 |
