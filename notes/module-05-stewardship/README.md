## 1. 尽责管理（Stewardship）

### 1.1 什么是尽责管理？

> **尽责管理**是投资者通过**监督和影响**被投资公司，促进公司长期价值创造和可持续发展的一套实践。

与 ESG 整合的定位区别：

| 概念 | 核心动作 | 时间点 |
|---|---|---|
| **ESG 整合** | 将 ESG 纳入**投资决策** | 投资前 / 持有分析中 |
| **尽责管理** | 通过**积极行动**影响公司 | 投资持有后 |

### 1.2 PRI 定义的尽责管理三大工具

| 工具 | 说明 |
|---|---|
| **参与（Engagement）** | 与被投资公司进行直接或间接沟通 |
| **投票（Voting）** | 在股东大会上行使投票权 |
| **升级（Escalation）** | 当参与无效时采取更强硬的措施 |

### 1.3 为什么尽责管理重要？

- **让 ESG 分析产生实际影响**：发现了 ESG 问题不行动，等于没有意义
- **降低系统性风险**：通过推动全行业改善 ESG 标准，减少整个投资组合的系统性风险
- **履行受托责任**：在许多法域，积极行使所有权已被视为受托责任的组成部分
- **长期价值创造**：改善公司 ESG 表现 → 降低风险、提升竞争优势 → 更好的长期回报

## 2. 尽责管理守则（Stewardship Code）⭐

### 2.1 国际背景

| 守则/框架 | 年份 | 管辖范围 |
|---|---|---|
| **UK Stewardship Code** ⭐ | 2010（2020 修订） | 英国（全球首个，影响力最大） |
| **日本 Stewardship Code** | 2014 | 日本 |
| **欧盟股东权利指令 II**（SRD II） | 2017 | 欧盟 |
| **ICGN 全球尽责管理原则** | 常更新 | 全球（非强制） |

### 2.2 UK Stewardship Code 2020 核心原则（12 条）

UK Stewardship Code 是全球标杆，2020 版强调**成果导向**（证明尽责管理产生了实际效果）：

**原则 1–5：目的与治理**
1. 公开披露尽责管理政策
2. 管理与利益冲突有关的尽责管理风险
3. 监控被投资公司
4. 制定清晰的参与升级策略
5. 必要时与其他投资者协作

**原则 6–8：投资方法**
6. 有明确的投票政策并披露投票活动
7. 定期报告尽责管理活动及其结果

**原则 9–12：参与**
9. 对资产所有者客户保持透明
10. 将 ESG 因素整合到尽责管理
11. 对固定收益和其他资产类别实施尽责管理
12. 披露尽责管理的具体成果

> **⚠️ 考试提示：** UK Stewardship Code 是"遵守或解释"（Comply or Explain），不是强制性的。

### 2.3 SRD II（欧盟股东权利指令）

| 要求 | 说明 |
|---|---|
| **机构投资者透明度** | 需公开披露参与政策和投票记录 |
| **资产管理人透明度** | 需向资产所有者报告尽责管理实施情况 |
| **Say-on-Pay** | 股东对董事薪酬有表决权 |
| **关联交易审查** | 重要的关联交易须经股东批准 |

## 3. 参与（Engagement）⭐

### 3.1 参与的类型

| 类型 | 说明 | 例子 |
|---|---|---|
| **个体参与**（Individual Engagement） | 单个投资者与公司直接沟通 | 基金经理与公司董事会面讨论碳减排目标 |
| **集体参与**（Collective Engagement） | 多个投资者联合与公司沟通 | Climate Action 100+ 集体与排放巨头对话 |
| **服务商主导参与**（Service Provider-led） | 通过尽责管理服务商（如 EOS at Federated Hermes）间接参与 | 小型投资者委托专业机构出面沟通 |

### 3.2 参与渠道

| 渠道 | 强度 | 说明 |
|---|---|---|
| 致函管理层/董事会 | 低 | 表达关切，请求回复 |
| 与管理层面谈 | 中 | 一对一或小组会议 |
| 与董事会（包括独立董事）面谈 | 高 | 更正式、更高层的对话 |
| 提交股东提案/决议 | 高 | 正式要求股东投票 |
| 公开声明/媒体 | 最高 | 施加公众压力 |

### 3.3 参与的升级策略（Escalation）

当初步参与不奏效时，投资者可以逐步升级：

```
第1级：与管理层进行私下对话
  ↓ 无实质进展
第2级：致函董事会/审计委员会
  ↓ 无实质进展
第3级：在股东大会上投反对票（反对董事连任、薪酬报告等）
  ↓ 无实质进展
第4级：提交或支持股东提案
  ↓ 无实质进展
第5级：公开声明/撤资（Divestment）
```

> **💡 撤资是尽责管理失败后的最后手段。** 撤资意味着放弃了影响公司的一切机会。

## 4. 代理投票（Proxy Voting）⭐⭐

### 4.1 投票机制

| 概念 | 说明 |
|---|---|
| **代理投票**（Proxy Voting） | 股东委托他人（通常是管理层或第三方）代为投票 |
| **代理声明**（Proxy Statement） | 股东大会前发给股东的文件，包含待表决事项的说明 |
| **代理顾问**（Proxy Advisor） | 提供投票建议的专业机构（如 ISS、Glass Lewis） |
| **投票记录** | UK Stewardship Code 等要求投资者披露其投票活动 |

### 4.2 股东大会上常见的 ESG 相关投票事项

| 投票事项 | ESG 相关性 |
|---|---|
| **董事选举/连任** | 投票反对 ESG 表现不力的董事（尤其审计委员会和薪酬委员会） |
| **薪酬报告** | Say-on-Pay — 薪酬方案是否与 ESG 业绩挂钩 |
| **股东提案** | 气候披露、DEI、政治献金透明度等议题 |
| **审计师任命** | 审计独立性和质量 |
| **公司章程修订** | 可能涉及投票权变化、反收购条款等治理议题 |

### 4.3 代理顾问（Proxy Advisors）的角色

| 角色 | 说明 |
|---|---|
| **提供研究** | 对股东大会议题进行分析和投票建议 |
| **争议** | 可能出现"一刀切"的建议；机构投资者可能过度依赖而放弃独立判断 |
| **监管动态** | SRD II 要求代理顾问披露其研究方法和利益冲突 |

## 5. 集体参与（Collective Engagement）

### 5.1 优势

- **增强影响力**：多个投资者的联合筹码比单打独斗更有效
- **分担成本**：参与需要时间和资源，集体行动费用共享
- **减少搭便车**：更多投资者参与 → 更公正地分担成本
- **降低监管风险**：在反垄断/一致行动规则下，正式化的集体参与更安全

### 5.2 法律考量：一致行动（Acting in Concert）

> 各国监管对投资者之间"一致行动"有不同的规则，集体参与可能触发达成要约收购等监管义务。因此，集体参与需在法律框架内审慎进行。

### 5.3 标志性集体参与倡议

| 倡议 | 聚焦议题 | 规模 |
|---|---|---|
| **Climate Action 100+** | 要求全球最大排放企业减排和披露 | 700+ 投资者，$68 万亿 AUM |
| **PRI 协作平台** | 推动签署方就特定 ESG 议题联合参与 | 全行业 |
| **Nature Action 100** | 与生物多样性最相关的公司的参与 | 新兴倡议 |

## 6. 不同资产类别的尽责管理

### 6.1 公开股权 vs 固定收益 vs 另类资产

| 资产类别 | 尽责管理工具 | 特点 |
|---|---|---|
| **公开股权**（Listed Equity） | 投票 + 参与 + 股东提案 | 权利最充分 |
| **固定收益**（Fixed Income） | 参与（尤其在一级市场） | 无投票权，但在债券合约中可加入 ESG 条款 |
| **私募股权**（Private Equity） | 董事会席位 + 直接控制 | 影响力最强但透明度最低 |
| **房地产/基建** | 直接运营控制 + 参与 | 运营层面的 ESG 管理 |

### 6.2 固定收益的尽责管理

债券持有人虽然没有投票权，但可以通过以下方式施加影响：
- **一级市场**：在债券发行前沟通 ESG 要求
- **绿色/可持续债券**的条款设计
- **债权人会议**上的发言权
- **信用分析**中纳入 ESG 因素并向发行人反馈

## 7. 衡量尽责管理的效果

### 7.1 效果评估指标

| 维度 | 指标 |
|---|---|
| **里程碑达成** | ESG 目标是否在规定时间内实现 |
| **公司行为变化** | 政策变更、董事会变动、披露改进 |
| **投票影响** | 股东提案获得支持率的变化趋势 |
| **长期财务表现** | 尽责管理是否带来更好的风险调整回报 |

### 7.2 挑战

- **归因困难**：如何证明公司改善是某个投资者的参与而非其他因素的结果
- **时滞：** ESG 改善需要数年才能体现为财务回报
- **保密约束**：有效的私下参与通常无法公开披露细节
