## 1. ESG 数据：来源与挑战

### 1.1 ESG 数据的层级

```
第1层：公司自身披露
  ├── 可持续发展报告、年报、CDP 问卷回复
  ├── 公司官网、投资者演示
  └── 优点：最直接；缺点：可能不完整，可能选择性披露

第2层：官方/监管数据
  ├── 公司注册信息、监管处罚记录
  └── 优点：客观、标准化；缺点：范围有限

第3层：第三方数据提供商
  ├── MSCI、Sustainalytics、ISS、Bloomberg ESG
  ├── 优点：覆盖面广、有方法论；缺点：方法论不透明、评级分歧大
  └── 多采用估计/建模来填补数据空白

第4层：替代数据 (Alternative Data)
  ├── 卫星图像（监测甲烷泄漏、森林砍伐）
  ├── NLP 分析新闻和社交媒体（声誉风险）
  └── 供应链追溯平台
```

### 1.2 ESG 数据的核心挑战 ⭐

| 挑战 | 说明 |
|---|---|
| **缺乏标准化** | 不同公司使用不同框架（GRI / SASB / TCFD）披露，指标不统一 |
| **覆盖不完整** | 中小企业、新兴市场、非上市公司普遍缺乏 ESG 数据 |
| **时效性差** | ESG 数据通常是年度披露，远慢于财务数据的季度频率 |
| **可靠性问题** | ESG 数据通常未经审计（assurance 有限），存在"漂绿"风险 |
| **向后看** | 披露的是过去数据，但 ESG 分析需要前瞻性视角 |
| **不可比性** | 同一行业的不同公司可能使用不同方法计算 Scope 3 排放 |
| **估计与推断** | 大量 ESG 数据由提供商通过模型估算而非实际数据 |

### 1.3 定量 vs 定性 ESG 数据

| 类型 | 例子 | 特点 |
|---|---|---|
| **定量** | 碳排放吨数、工伤率、董事会女性比例 | 可比较、可追踪趋势 |
| **定性** | ESG 政策质量、企业文化、社区关系案例 | 信息丰富但难以标准化比较 |

> **💡 最佳实践是定量 + 定性结合使用。** 只看数字可能忽略背景（"碳排下降"可能只是因为出售了高碳资产），只看文字无法跨公司比较。

## 2. ESG 评级 ⭐⭐

### 2.1 主要 ESG 评级提供商

| 评级商 | 方法论概述 |
|---|---|
| **MSCI ESG Ratings** | 评估行业特定的关键议题（Key Issues），给 CCC 到 AAA 评级 |
| **Sustainalytics**（晨星） | 以"未管理风险"（Unmanaged Risk）为核心，分数越低越好 |
| **ISS ESG** | 从 Prime（最佳）到 C-（最差） |
| **S&P Global ESG Scores** | 基于 CSA（企业可持续性评估）问卷 |
| **CDP Scores** | 专门评估环境披露和表现（气候变化、水、森林） |
| **FTSE Russell** | 纳入指数成分股管理的 ESG 评分 |
| **Bloomberg ESG** | 基于公开披露数据的评分 |

### 2.2 ESG 评级分歧（ESG Rating Divergence）⭐

> 这是 ESG 投资领域最重要的学术发现之一：**不同评级商对同一公司的 ESG 评分相关性很低**（平均相关系数仅 ~0.3–0.5，远低于信用评级 ~0.99）。

**分歧来源：**

| 来源 | 说明 |
|---|---|
| **范围分歧**（Scope Divergence） | 不同评级商关注的指标不同（Sustainalytics 重视风险，MSCI 重视机遇） |
| **度量分歧**（Measurement Divergence） | 同一概念用了不同的度量方式（如用绝对碳排放还是碳排放强度？） |
| **权重分歧**（Weight Divergence） | 同一指标被赋予不同权重（碳排放对某评级商是 30%，对另一个是 10%） |

**对投资者的影响：**
- 不能依赖单一评级来源
- 应理解评级背后的**方法论**而非只看评级结果
- 可以将多个评级取平均（但会丢失方法论信息）
- 最好的做法是自己做 ESG 分析，用评级做参考

### 2.3 如何使用 ESG 评级

| 用途 | 说明 |
|---|---|
| **筛选/排除** | 排除低于某评级阈值的公司 |
| **加权调整** | 对高评级公司增加组合权重 |
| **风险警示** | 评级大幅下降触发重新评估 |
| **参与优先** | 低评级但高改善潜力 → 作为参与目标 |

## 3. 重要性（Materiality）⭐

### 3.1 财务重要性 vs 双重重要性

| 概念 | 定义 | 代表框架 |
|---|---|---|
| **财务重要性**（Financial Materiality） | ESG 因素是否对**公司的财务表现**有显著影响 | SASB, ISSB |
| **双重重要性**（Double Materiality） | 除了 ESG 对公司的影响，也考虑**公司对社会和环境的影响** | GRI, 欧盟 CSRD |
| **动态重要性**（Dynamic Materiality） | 今天不重要的 ESG 议题可能随时间变得重要 | 所有框架 |

### 3.2 SASB 重要性地图（Materiality Map）

SASB 为 77 个行业分别确定了**对财务有实质影响的 ESG 议题**。

| 行业 | 实质性 ESG 议题举例 |
|---|---|
| 石油天然气 | 温室气体排放、生态影响、社区关系 |
| 半导体 | 产品生命周期管理、原材料采购、员工 DEI |
| 银行 | 系统性风险管理、客户隐私、普惠金融 |
| 食品饮料 | 食品安全、营养、水资源管理、包装废弃物 |
| 医疗 | 产品安全和定价、临床试验透明度 |

> **💡 重要性的核心判断：** 如果一个 ESG 信息被忽略会改变投资者的决策，那它就是"重要"的。

## 4. ESG 整合方法 ⭐⭐

### 4.1 将 ESG 纳入基本分析

| 步骤 | 说明 |
|---|---|
| **识别（Identify）** | 根据行业和商业模式，识别实质性 ESG 议题 |
| **分析（Analyze）** | 评估 ESG 对公司战略、财务、竞争地位的影响 |
| **调整（Adjust）** | 将 ESG 分析结果反映到估值模型假设中 |

### 4.2 ESG 对估值模型的调整方式

**在 DCF（现金流量折现）模型中调整：**

| 调整方法 | 具体做法 |
|---|---|
| **调整收入增长** | 绿色转型可能带来新收入来源（如新能源业务）、ESG 丑闻损害销售 |
| **调整运营利润** | 碳税增加成本、节能措施降低成本、DEI 提高人才留存 |
| **调整资本支出** | ESG 相关的设备更新/技术投入/合规投资 |
| **调整贴现率** | ESG 风险高 → 用更高的 WACC（加权平均资本成本） |
| **调整终值和增长假设** | 无法持续的公司不能假设长期高增长 |

**在相对估值（倍数法）中调整：**
- ESG 表现好的公司可能获得**估值溢价**（higher multiple）
- ESG 风险高的公司应有**估值折扣**（lower multiple）

### 4.3 ESG 调整的贴现率

| ESG 风险类型 | 对贴现率的影响 |
|---|---|
| 治理差（弱股东权利、关联交易） | 增加股权风险溢价 → 提高 COE |
| 气候转型风险高 | 增加系统性风险 → 提高 CAPM β |
| 环境负债（污染清理） | 增加违约风险 → 提高 COD |

## 5. 固收中的 ESG 分析 ⭐

### 5.1 信用分析中的 ESG 整合

固定收益投资者整合 ESG 的动机和股权投资者不同：
- 债券投资者的回报有**上限**（本息回收），但**下行风险**巨大（违约 = 本金全损）
- 因此固定收益 ESG 分析特别重视**下行风险识别**

| ESG 风险 | 对信用质量的影响 |
|---|---|
| 环境诉讼 | 罚款增加负债，现金流紧张 |
| 治理丑闻 | 融资成本上升、再融资困难 |
| 搁浅资产 | 公司核心资产贬值，担保覆盖率降低 |
| 供应链中断（社会因素） | 收入中断，无法支付利息 |

### 5.2 ESG 在固收中的特殊应用

| 应用 | 说明 |
|---|---|
| **信用评级中的 ESG 调整** | Moody's、S&P 等传统评级商已开始将 ESG 作为信用调整因素 |
| **绿色/可持续债券分析** | 评估债券融资项目的实际 ESG 影响 |
| **主权债 ESG** | 评估国家的治理质量、环境政策、社会稳定性 |

## 6. ESG 分析的挑战与最佳实践

### 6.1 关键挑战

| 挑战 | 具体表现 |
|---|---|
| **时间范围不匹配** | ESG 影响往往在长期（5-10年+）体现，但投资考核周期以季度/年计 |
| **回测数据不足** | ESG 数据历史短（多数仅 5-10 年），难以做统计上可靠的量化分析 |
| **相关性 ≠ 因果关系** | "ESG 好 = 回报高"可能是好公司同时在财务和 ESG 上表现出色，而非 ESG 导致了好回报 |
| **行业差异大** | 同样的 ESG 指标在不同行业中意义完全不同 |

### 6.2 最佳实践

- **从行业出发**：先识别该行业最实质的 ESG 议题，再有针对性地收集数据
- **多来源交叉验证**：不要依赖单一 ESG 数据或评级来源
- **前瞻性分析**：不止看当前 ESG 数据，更要分析 ESG 趋势和公司应对策略
- **定量+定性结合**：数字给出基准，定性判断给出深度
